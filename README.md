# House Price Prediction

This project predicts house prices using Machine Learning techniques. 
The model is trained on a housing dataset to estimate the price of houses based on different features.

## Dataset
The dataset contains information about houses such as area, number of rooms, location features, and price.

## Libraries Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Model
Linear Regression

## Steps
1. Data preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature selection
4. Model training using Linear Regression
5. Model evaluation and prediction
